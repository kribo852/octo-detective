# octo-detective, a 2d Detective game
It would be fun to make a little detective game, a little project of mine

## The game so far 
You play as a detective, and try to solve crimes. Investigate the crime scene and see if you can solve the case. 
My intention is to have as much of the configuration as possible in the level files, so that creators can create their own little crime mysteries. But the game is not so complete yet, as to do that. 


## Technical information
Built using Love 2D version 11.5
